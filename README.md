# 💖 Nossa História — Projeto Web Interativo

Este é um projeto de página web interativa desenvolvido com **HTML, CSS e JavaScript**, criado com o objetivo de representar uma linha do tempo de um relacionamento de forma visual, dinâmica e emocional.

## 🚀 Funcionalidades

O projeto vai além de uma página estática, incluindo diversas interações e animações:

### ✨ Interface e Experiência

* Animação de background com gradiente dinâmico
* Layout centralizado em formato de “card”
* Design responsivo para mobile

### 💬 Interações

* Efeito **typewriter** na mensagem principal
* Botão para reiniciar a animação
* Sistema de **carrossel de imagens automático**
* Navegação por pontos no carrossel

### 📊 Sistema de tempo

* Contador automático de dias:

  * Desde o primeiro beijo
  * Desde o início do namoro

### 📖 Linha do tempo (Timeline)

* Exibição de momentos importantes
* Estrutura alternada (esquerda/direita)
* Suporte a imagens e descrições
* Animação de entrada sequencial

### 🎁 Vales interativos

* Cartões clicáveis
* Sistema de “resgate” com feedback visual
* Mudança de estado (ativo → resgatado)

### 💕 Motivos aleatórios

* Botão que gera frases dinâmicas
* Animação suave de transição

### ❤️ Animações

* Corações flutuantes contínuos
* Interação ao clicar nas fotos de perfil

### 🥚 Easter Egg

* Digite **"te amo"** no teclado
* Ativa uma **chuva de corações animada**
